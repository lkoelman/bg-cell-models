# Installation


## Install useful tools

### nbstripout

```bash
pip install --upgrade nbstripout
cd bgcellmodels
nbstripout install
```

### ipython extensions

```bash
pip install jupyter_contrib_nbextensions
jupyter nbextensions_configurator enable --user
```

Now extensions can be enabled via Jupyter extensions tab
